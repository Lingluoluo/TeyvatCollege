### 我也想搞一个怎么办

1. 购买域名，购买服务器，进行域名解析，在NGINX中添加网站 (地址位于中国大陆的服务器还需要进行备案)
1. clone本项目
1. 自行替换`src`文件夹中的`huc.jpg`
1. 修改`index.html` `index-mobile.html`中的网页标题
